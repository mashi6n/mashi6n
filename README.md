## Repositories
### [kyounoryouri-tools](https://github.com/mashi6n/kyounoryouri-tools)
- A collection of utilities to download and extract recipe data from Minna no Kyō no Ryōri, a cooking recipe website operated by Japan Broadcasting Corporation (NHK).
- Related to the paper "[A Highly Clean Recipe Dataset with Ingredient States Annotation for State Probing Task](https://doi.org/10.1145/3746027.3755771)" accepted at ACMMM 2025.

### [cudep-web](https://github.com/mashi6n/cudep-web)
- A simple web application that helps you resolve dependencies for CUDA, GPUs and GPU drivers.
